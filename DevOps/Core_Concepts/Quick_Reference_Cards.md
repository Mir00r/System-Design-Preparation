# **Quick Reference Cards** 🎴

---

## **🎯 How to Use These Cards**

```
Study Method:
  1. Print double-sided (Front = Question, Back = Answer)
  2. Cut along dotted lines
  3. Study daily (15 minutes minimum)
  4. Shuffle and test yourself
  5. Separate into "Know" and "Review" piles
  6. Focus on "Review" pile daily

Spaced Repetition:
  Day 1: Learn new cards
  Day 2: Review Day 1 cards
  Day 4: Review again
  Day 7: Review again
  Day 14: Review again
  Day 30: Review again

Print Instructions:
  - Use cardstock for durability
  - Print double-sided (flip on short edge)
  - Laminate for long-term use
  - Hole punch and use ring for organization
```

│                                         │

│ Imperative = Recipe steps               │
│ Declarative = Order from menu           │

### **Card 15**
```
┌─────────────────────────────────────────┐
│ FRONT                                   │
├─────────────────────────────────────────┤
│                                         │
│  SLI, SLO, SLA?                         │
│  (Definitions + Example)                │
│                                         │
└─────────────────────────────────────────┘
┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄
┌─────────────────────────────────────────┐
│ BACK                                    │
├─────────────────────────────────────────┤
│ SLI (Service Level Indicator):         │
│ Actual measurement of service level     │
│ Example: 99.95% availability measured   │
│                                         │
│ SLO (Service Level Objective):          │
│ Target value for SLI (internal goal)    │
│ Example: Target 99.9% availability      │
│                                         │
│ SLA (Service Level Agreement):          │
│ Contract with customer (consequences)   │
│ Example: Guarantee 99.5%, refund if not │
│                                         │
│ Error Budget:                           │
│ 100% - SLO = Allowed downtime           │
│ Example: 0.1% = 43 min/month            │
└─────────────────────────────────────────┘
```

### **Card 16**
```
┌─────────────────────────────────────────┐
│ FRONT                                   │
├─────────────────────────────────────────┤
│                                         │
│  Calculate Error Budget                 │
│  SLO: 99.9% uptime                      │
│  Time: 30 days                          │
│                                         │
└─────────────────────────────────────────┘
┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄
┌─────────────────────────────────────────┐
│ BACK                                    │
├─────────────────────────────────────────┤
│ Formula:                                │
│ Error Budget = (1 - SLO) × Total Time   │
│                                         │
│ Calculation:                            │
│ SLO = 99.9% = 0.999                     │
│ Error Budget = (1 - 0.999)              │
│              = 0.001 (0.1%)             │
│                                         │
│ Time in minutes:                        │
│ 30 days × 24 hours × 60 min = 43,200 min│
│                                         │
│ Error Budget in minutes:                │
│ 0.001 × 43,200 = 43.2 minutes           │
│                                         │
│ Answer: 43 minutes downtime allowed!    │
└─────────────────────────────────────────┘
```
