# 📅 90-Day Interview Prep Plan: Comprehensive Mastery

> *"90 days is the gold standard for FAANG preparation. You have time to build genuine mastery — not just memorize solutions, but develop intuition that lets you solve problems you've never seen before."*

**⏱️ Estimated Time**: 10 minutes | **🎯 Difficulty**: 🟡 Medium

---

## 📋 Phase Overview

```
MONTH 1 (Days 1-30):   LEARN — Build strong foundations
MONTH 2 (Days 31-60):  PRACTICE — Apply knowledge, build speed
MONTH 3 (Days 61-90):  PERFORM — Mock interviews, refine, peak

DAILY COMMITMENT: 2-3 hours (sustainable over 3 months)
  Coding:     1 hr (2 problems)
  Design:     45 min
  Behavioral: 15 min
  + Weekend deep dives (3-4 hrs)
```

---

## 📅 Month 1: Foundation Building

```
WEEK 1: Arrays, Strings, HashMap
  Problems: Two Sum, 3Sum, Group Anagrams, Longest Substring,
            Valid Anagram, Product of Array Except Self
  Design: Read Caching, Load Balancing, CAP Theorem
  
WEEK 2: Two Pointers, Sliding Window, Binary Search
  Problems: Container With Water, Minimum Window Substring,
            Search Rotated Array, Find Minimum, Median of Two Arrays
  Design: Read Database types, Sharding, Indexing
  
WEEK 3: Linked Lists, Stacks, Queues
  Problems: Reverse LL, Merge K Lists, LRU Cache,
            Valid Parentheses, Daily Temperatures, Sliding Window Max
  Design: Read Message Queues, API Gateway, Rate Limiting
  Practice: Design URL Shortener (first attempt)
  
WEEK 4: Trees (BFS, DFS, BST)
  Problems: Max Depth, Validate BST, LCA, Serialize Tree,
            Binary Tree Level Order, Kth Smallest in BST
  Design: Read Consistent Hashing, Replication, Consensus
  Practice: Design Twitter Feed (first attempt)

MONTH 1 TARGET: 50-60 problems solved, understand all building blocks
```

---

## 📅 Month 2: Pattern Mastery

```
WEEK 5: Graphs (BFS, DFS, Topological Sort)
  Problems: Number of Islands, Clone Graph, Course Schedule,
            Word Ladder, Pacific Atlantic, Alien Dictionary
  Design: Design WhatsApp (real-time messaging)
  
WEEK 6: Heap, Trie, Union-Find
  Problems: Top K Frequent, Merge K Lists, Find Median,
            Implement Trie, Word Search II, Accounts Merge
  Design: Design Netflix (streaming + recommendation)
  
WEEK 7: Dynamic Programming
  Problems: Climbing Stairs, Coin Change, LCS, Edit Distance,
            Word Break, Longest Palindromic Substring, House Robber
  Design: Design Uber (real-time location, matching)
  START MOCKS: 1 mock/week (coding)
  
WEEK 8: Backtracking, Greedy, Intervals
  Problems: Permutations, N-Queens, Combination Sum,
            Jump Game, Task Scheduler, Merge Intervals
  Design: Design Payment System (transactions, consistency)
  MOCKS: 1 coding mock + 1 design mock

MONTH 2 TARGET: 100-120 problems total, 4 system designs practiced,
                4 mock interviews completed
```

---

## 📅 Month 3: Peak Performance

```
WEEK 9: Hard Problems + Weak Areas
  Revisit patterns you struggled with (spaced repetition)
  Practice 1 Hard problem per day
  Design: Design Google Search / Notification System
  MOCKS: 2 per week (mix coding + design)

WEEK 10: System Design Deep Dives
  Design: Design Dropbox, Design YouTube, Design Instagram
  Review all designs — focus on trade-offs, not just architecture
  Behavioral: Refine all 10 STAR stories
  MOCKS: 3 per week

WEEK 11: Full Interview Simulation
  Day 1: Full FAANG loop simulation (4 rounds, 1 day)
  Day 2: Review and fix gaps
  Day 3-4: Focus on identified weaknesses
  Day 5: Another full simulation
  Company-specific prep (read culture, blog posts, products)
  MOCKS: 3-4 per week

WEEK 12: Final Polish
  Light coding (maintain skills, don't cram)
  Review notes, patterns, system design summaries
  Prepare questions for YOUR interviewers
  Rest, exercise, confidence building
  TRUST YOUR PREPARATION

MONTH 3 TARGET: 150-180 problems total, 10+ system designs,
                15-20 mock interviews completed, ready to perform
```

## 🎯 What Makes 90 Days Special

```
1. SPACED REPETITION: Re-solve problems at day 7, 14, 30
   → Permanent retention vs cramming that fades

2. GENUINE MASTERY: Enough reps to develop INTUITION
   → See problem → instantly recognize pattern (no thinking required)

3. MOCK INTERVIEW VOLUME: 15-20 mocks
   → Interview anxiety disappears after mock #10

4. SUSTAINABLE PACE: 2-3 hrs/day doesn't burn out
   → Arrive at interview rested and confident, not exhausted

5. DEEP UNDERSTANDING: Time to read DDIA, watch lectures, discuss
   → Can answer "why" not just "what"
```
