## 🏗️ The Three Pillars of Technical Interviews

```
┌─────────────────────────────────────────────────────────────────┐
│                TECHNICAL INTERVIEW PILLARS                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. 💻 CODING (Data Structures & Algorithms)                    │
│     - Problem solving ability                                   │
│     - Code quality and communication                            │
│     - Time/space complexity analysis                            │
│                                                                 │
│  2. 🏛️ SYSTEM DESIGN                                            │
│     - Scalability and distributed systems                       │
│     - Trade-off analysis                                        │
│     - Communication and collaboration                           │
│                                                                 │
│  3. 🤝 BEHAVIORAL                                               │
│     - Leadership and teamwork                                   │
│     - Conflict resolution                                       │
│     - Growth mindset and self-awareness                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

WEIGHT BY LEVEL:
  Junior:  70% Coding, 10% Design, 20% Behavioral
  Mid:     50% Coding, 30% Design, 20% Behavioral
  Senior:  30% Coding, 40% Design, 30% Behavioral
  Staff+:  20% Coding, 50% Design, 30% Behavioral
```
