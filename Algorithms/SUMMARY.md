### **1. Sorting Algorithms** (8 algorithms)

**File**: [Sorting/01_Sorting_Fundamentals.md](./Sorting/01_Sorting_Fundamentals.md)

```
Simple O(n²):
• Bubble Sort       - Teaching friendly
• Selection Sort    - Minimum swaps
• Insertion Sort    - Good for small/nearly sorted

Efficient O(n log n):
• Merge Sort        - Guaranteed performance, stable
• Quick Sort        - Fastest in practice
• Heap Sort         - In-place O(n log n)

Specialized:
• Counting Sort     - O(n+k) for integers
• Radix Sort        - O(d×n) for fixed digits
```

## 📊 Complexity Quick Reference

### **Common Time Complexities**

```
O(1)       Constant      - Hash table lookup
O(log n)   Logarithmic   - Binary search
O(n)       Linear        - Linear search
O(n log n) Linearithmic  - Merge sort, Quick sort
O(n²)      Quadratic     - Bubble sort, Nested loops
O(2ⁿ)      Exponential   - Recursive Fibonacci
O(n!)      Factorial     - Generate permutations
```

### **Algorithm Complexity Table**

| Algorithm | Time | Space | Use Case |
|-----------|------|-------|----------|
| Binary Search | O(log n) | O(1) | Sorted array search |
| Merge Sort | O(n log n) | O(n) | Guaranteed performance |
| Quick Sort | O(n log n) avg | O(log n) | General purpose |
| Dijkstra | O((V+E) log V) | O(V) | Shortest path |
| DFS/BFS | O(V+E) | O(V) | Graph traversal |
| DP (typical) | O(n²) | O(n) | Optimization |
